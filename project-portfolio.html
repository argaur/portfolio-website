Constraint

Why No Framework

The constraint was intentional: no npm, no React, no Tailwind, no build step. The site must work by opening index.html in a browser. This was both a technical demonstration (can you ship a polished product with nothing but platform primitives?) and a practical choice (no dependency rot, no build failures, instant iteration).

The result: 0 npm dependencies. The only external dependency is the Supabase JS client loaded via CDN for the email gate.

Design System

Blueprint to Bits

The design theme is called Blueprint to Bits — a reference to the career arc from architectural drawings to digital products. The visual language uses technical drawing conventions: grid paper backgrounds, no rounded corners, serif display type, and a restrained palette.

Design Tokens

#0b1628

--navy — Primary dark. Used for hero, contact, nav, and dark UI surfaces.

#f7f3ee

--cream — Primary light. Used for content panels and grid-paper backgrounds.

#bf5c3a

--terra — Terracotta accent. Used for CTAs, category labels, and active states.

Typography

  • Cormorant Garamond — headings, display text, large numbers (editorial, high contrast weight range)
  • DM Sans — body text, UI labels, navigation (clean, geometric, legible at small sizes)

Grid Background

The graph-paper grid is a pure CSS trick using two overlapping linear-gradient backgrounds at 120px spacing. No SVG, no image. Applied to all cream and white panels at rgba(0,0,0,0.06) opacity.

No Border-Radius

Every element is square-cornered. This is an explicit architectural principle — the design language of technical drawings has no rounded corners. Softness comes from spacing and type weight, not corner radius.

Architecture

Panel-Based SPA

The site is a single HTML file with 7 full-viewport panels. Only one panel is visible at a time. Navigation switches panels by toggling an is-active class — no page loads, no scroll, no router library.

7 Panels

  • Home — hero, blueprint grid, statement, single CTA to Contact
  • Experience — career arc bar, 4 accordion rows (Architecture → Founding → Product → AI)
  • Work — 4 company groups, 9 project cards, click-to-modal details
  • Projects — 5 PM case studies + 4 personal projects, unified modal
  • Skills — navy panel, 3-col skill groups across 6 domains
  • Credentials — split panel (education left / 11 certifications right)
  • Contact — navy + blueprint grid, email + LinkedIn links

Hash Routing

On page load, app.js reads window.location.hash and activates the matching panel. This means every panel has a shareable direct URL: #experience, #work, #projects, etc.

Mobile Nav

The desktop nav collapses into a hamburger button on mobile. Tapping it opens a full-screen overlay with numbered Cormorant Garamond links, left-aligned with generous padding. ESC key or selecting a panel closes it.

Email Gate

Lead Capture via Supabase

First-time visitors see a full-screen gate before the portfolio. They enter an email address, which is inserted into a Supabase subscribers table. On success, a localStorage flag is set so returning visitors skip the gate entirely.

localStorage bypass flag for returning visitors
Graceful degradation if Supabase is unreachable

The anon key is embedded in gate.js — safe for client-side use because RLS is configured to allow anonymous inserts only. No read or delete access from the client.

File Structure

What's in the Repo

index.html

All 7 panels, modal overlay, mobile nav overlay. Single file, no partials, no templating.

styles.css

Blueprint to Bits design system (~1,350+ lines). Panel layout, all section styles, mobile responsive breakpoints.

app.js

Panel switching, mobile overlay nav, experience accordion, work project modal (9 projects), unified item modal (case studies + personal projects).

gate.js

Email gate logic: Supabase insert, localStorage flag, graceful degradation on error.

case-study.css

Shared styles for all case study and project detail pages. Pagination pill, section layout, callout boxes.

favicon.svg

Geometric GG monogram — navy background, white letterforms, terracotta crossbars. Pure SVG, no raster.
