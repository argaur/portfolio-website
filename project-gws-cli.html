GWS CLI

A Python command-line tool for Gmail and Google Calendar — search, draft, and schedule without opening a browser. Telegram trigger integration in progress.

Python Gmail API Google Calendar API CLI
What It Is

Google Workspace from the Terminal

GWS CLI (Google Workspace CLI) is a Python tool that wraps the Gmail and Google Calendar APIs with a command-line interface. The goal: handle the most common Gmail and Calendar tasks without switching to a browser, without navigating a UI, and without context-switching out of the terminal.

The interface is intentionally thin — thin wrappers over the Google APIs with clean output. No heavy abstraction, no config files required beyond OAuth credentials.

Capabilities

What It Can Do

Gmail

Search threads, read messages, create drafts, and manage labels from the command line.

Google Calendar

List upcoming events, create events, and check availability — without opening a browser tab.

Next: Telegram Trigger

The planned next step is connecting GWS CLI to the Telegram Personal Assistant Bot. This would allow triggering Gmail and Calendar operations via a Telegram message — the same single-interface pattern used for other automation tasks. For example: checking tomorrow's calendar without unlocking a laptop.

Next: rclone Integration

rclone integration is planned to enable syncing and managing Google Drive files from the same CLI, extending GWS CLI from a communication tool to a broader workspace tool.
