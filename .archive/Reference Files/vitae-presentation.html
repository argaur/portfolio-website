Technical Architecture

Production-grade stack. Shipped in 10 days.

⚛️

Frontend

Next.js 16.2 · React 19 · TypeScript strict

App Router, Server Components, Server Actions, Tailwind CSS v4, PWA with service worker.

🗄️

Backend / Database

Supabase · PostgreSQL · RLS

Row-level security on every table. Email + Google OAuth. Supabase Storage for prescription files.

🔍

OCR Engine

Gemma 4 26B · OpenRouter · 256K context

Multimodal — reads handwritten prescriptions directly from images. Extracts doctor, date, medications, dosages with confidence scores.

🧠

AI Explanation

Claude (Anthropic) · Free model pool

Structured prescription data → plain-language explanation per medication. 429 fallback across free model pool.

🚀

Deployed on Vercel — auto-deploy from master branch. Live at health-assistant-snowy.vercel.app

AI Layer

Two AI pipelines. One seamless flow.

Pipeline 1 — OCR Extraction

→

Gemma 4 26B reads prescription image directly (multimodal)

→

Extracts: doctor name, date, medication names, dosage, frequency, duration

→

Assigns confidence: high / low per field

→

Same pipeline handles lab reports — extracts test values, units, reference ranges, flags abnormals

Pipeline 2 — Plain-Language Explanation

→

Claude takes structured prescription data, generates 4-dimension explanation per medication

→

Dimensions: treats · how to take · side effects · things to avoid

→

Generates "Things to tell your doctor" actionable list

→

Explanation cached in DB — never re-generated on repeat visits

Example Output

Metformin 500mg · BD

Treats: Type 2 diabetes — helps lower blood sugar after meals

How to take: With meals, morning and night

Side effects: Nausea (common, usually fades)

Avoid: Alcohol, heavy carbs in one sitting

Atorvastatin 10mg · OD HS

Treats: High cholesterol — reduces LDL ("bad") cholesterol

How to take: At bedtime with water

Side effects: Muscle aches (rare)

Avoid: Grapefruit juice
