The Manifesto for Strategic Logic

01

Purpose Before Aesthetics

Define what the artifact must prove before any visual choice is made.

02

Systems Thinking First

Understand the whole before optimising any part. Everything is a node in a larger network.

03

Complexity Is Failure

If users need training, the design failed first. Strip away noise until only the essential remains.

04

Build For the Constraint

Constraints reveal the real design problem, not the ideal. Use them, don't fight them.

05

Medium Is the Message

The artifact itself should demonstrate the skill. Every surface is a proof point.

06

Ship to Learn

A hypothesis is only as good as the data that refutes or confirms it. Iterate relentlessly.

07 [CAPSTONE]

Inevitability Test

If the design language could be transplanted to a different product without feeling wrong, it was imposed, not derived. Every choice must be earned.
